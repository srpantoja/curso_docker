# Curso Docker

## subir docker-compose
`docker-compose up` 


`docker-compose up -d` 

## derrubar docker-compose
`docker-compose down` 


`docker-compose down -v` 

## visualizar logs
`docker-compose logs -f -t` 

## criar multiplas instâncias
`docker-compose up -d --scale worker=3`
